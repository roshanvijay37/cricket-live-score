# Cricket Live Score MVP

A simple cricket live score application built with .NET Web API backend and React frontend.

## Project Structure

```
cricket-live-score/
├── backend/CricketAPI/     # .NET Web API
├── frontend/cricket-app/   # React Application
└── README.md
```

## Running the Application

### Backend (.NET API)
```bash
cd backend/CricketAPI
dotnet run
```
API will run on: https://localhost:7000

### Frontend (React)
```bash
cd frontend/cricket-app
npm start
```
Frontend will run on: http://localhost:3000
